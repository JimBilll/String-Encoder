Gödel Word Encoder

By James McLean



import js import frontend outArea = Element("outputArea") def encodeText(): if (Element("textField").value != ''): text = Element("textField").value outArea.element.innerHTML = frontend.readText(text) def decodeText(): if (Element("codeField").value != ''): text = Element("codeField").value outArea.element.innerHTML = frontend.readCode(text)

Note: If you try to decode a random Godel number there's a good chance it will give a strange answer, this is because it is generating sequences which do not correspond to valid text. In some cases it could also be because it requires primes higher than those which are generated, however any word you can encode should be decodable.
